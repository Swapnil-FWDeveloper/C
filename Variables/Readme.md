## Variables

- **Definition**: A variable is the name of a memory location which stores some data.

- **Syntax**:
  ```python
  variable_name = data

data_type: The type of data the variable will store (e.g., int, float, char).
variable_name: The name you give to the memory location.
value: The initial value assigned to the variable.

## Rules
a. Variables are case sensitive
b. 1st character is alphabet or "_"
c. no comma/blank space
d. No other symbol other than "_"

## Variable Data Type Size in Bytes

+---------------------+---------------------+----------------------+
| Data Type           | Description         | Size (in Bytes)      |
+---------------------+---------------------+----------------------+
| int                 | Integer             | 4B                   |
+---------------------+---------------------+----------------------+
| short               | Short integer       | 2B                   |
+---------------------+---------------------+----------------------+
| long                | Long integer        | 4B                   |
+---------------------+---------------------+----------------------+
| long long           | Long long integer   | 8B                   |
+---------------------+---------------------+----------------------+
| float               | Single-precision    | 4B                   |
|                     | floating point      |                      |
+---------------------+---------------------+----------------------+
| double              | Double-precision    | 8B                   |
|                     | floating point      |                      |
+---------------------+---------------------+----------------------+
| char                | Character           | 1B                   |
|                     |                     | or 1B                |
+---------------------+---------------------+----------------------+
| _Bool               | Boolean             | 1B                   |
+---------------------+---------------------+----------------------+
